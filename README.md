A personal developer portfolio website created to showcase my software development projects, technical skills, and contact information. The website highlights projects related to Java development, web development, and database systems.

## Project Overview

This portfolio was designed to present my work in a clean and professional format. It features a responsive layout, modern UI styling, and project sections that provide descriptions and links to GitHub repositories.

The website demonstrates front-end development and back-end development skills.

## Features

- Responsive portfolio layout
- Project showcase with screenshots and descriptions
- GitHub repository links for each project
- Live demo buttons for selected projects
- Professional footer with social and tech stack icons
- Contact page for professional inquiries
- Modern UI styling using CSS

### Sliding Block Puzzle Game

Technologies:
- Java
- JDBC
- Sqlite

### People's Bank Web App
A FinTech-style web banking interface developed using HTML, CSS, and JavaScript. The application includes financial planning tools such as a savings planner and loan calculator, demonstrating real-time calculations and modern interactive UI design.
